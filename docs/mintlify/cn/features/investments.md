---
title: "投资"
description: "锁定物品，赚取年利率，稍后赎回。"
---

投资让玩家将持仓中的物品锁定到类似证书的仓位中，随时间累积**年利率**。

## 工作原理

1. 玩家投资一定数量的材料。
2. 物品从流动持仓中移除。
3. 随时间推移，投资累积利息。
4. 玩家赎回并收回原始物品（到持仓）加上利息（通常以硬币支付）。

## 配置（典型）

```yaml
investments:
  enabled: true
  apr-percent: 5.0
```

## 命令（典型）

```text
/market invest buy <material> <amount>
/market invest list
/market invest redeem-all
```

<Warning>
如果您的服务器将赎回价值与当前市场价值挂钩，投资并非没有风险；您投资期间价格可能会变动。
</Warning>

## 年利率计算

利息根据年利率 (APR) 累积：

```
价值 = 本金 × (1 + APR × (持有天数 / 365))
```

### 示例

- 投资：100 个钻石，每个 $100 = $10,000
- APR：5%
- 持有时间：30 天

```
利息 = $10,000 × 0.05 × (30 / 365) = $41.10
总价值 = $10,041.10
```

---

## 到期

投资在赎回前有**最短持有时间**：

```yaml
investments:
  min-hold-time: 3600  # 秒（1 小时）
```

- **到期前** — 无法赎回
- **到期后** — 可随时赎回，继续赚取

---

## 配置

```yaml
investments:
  # 启用投资系统
  enabled: true
  
  # 年利率 (5.0 = 5%)
  apr: 5.0
  
  # 赎回前的最短持有时间（秒）
  min-hold-time: 3600
  
  # 每个玩家的最大投资数
  max-per-player: 10
  
  # 每笔投资的最大价值
  max-value: 100000
```

## 投资策略

### 长期持有

更高的 APR 意味着更长的持有更有利可图：

| 持有时间 | 5% APR 回报 |
|-----------|---------------|
| 1 天 | +0.014% |
| 7 天 | +0.096% |
| 30 天 | +0.411% |
| 90 天 | +1.233% |
| 365 天 | +5.000% |

### 价格投机

投资锁定您的购买价格。如果市场价格上涨：

- 您的投资价值反映更高的价格
- 赎回时获得比投资更多的硬币

<Warning>
如果市场价格下跌，您的赎回价值也会下降。投资并非没有风险！
</Warning>

---

## 赎回

赎回时：

1. 物品返回到您的**持仓**（不是背包）
2. 您收到累积价值的硬币
3. 投资关闭

### 您获得什么

```
赎回 = (原始物品) + (利息硬币)
```

示例：
- 投资：100 个钻石，$100 = $10,000
- 赚取的 APR：$200
- **收到：** 100 个钻石 + $200 硬币

## 技巧

<Tip>
**分散投资**  
不要把所有财富放在一个投资中。分散到多种材料。
</Tip>

<Info>
**关注市场**  
投资您认为会涨价的物品以获得双重收益（APR + 价格上涨）。
</Info>

<Warning>
**流动性**  
投资的物品被锁定。不要投资您可能很快需要的物品。
</Warning>
