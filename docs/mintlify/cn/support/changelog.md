## 升级说明

### 从 v1.4.x 升级到 v1.5.x

1. 备份 `plugins/TheEndex/` 文件夹
2. 停止服务器
3. 替换插件 JAR
4. 启动服务器
5. 检查控制台是否有迁移消息
6. 验证数据完整性

### 从 v1.5.x 升级到 v1.5.7

1. 停止服务器（或使用热重载）
2. 替换插件 JAR
3. 运行 `/endex reload`
4. 网页仪表盘将自动获得新的翻译功能

<Warning>
**始终备份**  
升级前始终备份您的数据。
</Warning>
