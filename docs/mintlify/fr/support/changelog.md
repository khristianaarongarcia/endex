# Journal des Modifications (Changelog)

## v1.5.7-dec1022 (Version Actuelle)
- **Nouveau** : Support multi-langue pour le tableau de bord web.
- **Nouveau** : Système de traduction automatique des fichiers de configuration.
- **Amélioration** : Optimisation des performances de la base de données.
- **Correctif** : Résolution d'un bug où les prix ne se sauvegardaient pas lors de l'arrêt brutal.

## v1.5.0
- **Nouveau** : Ajout du système de [Portefeuille (Holdings)](/fr/features/holdings).
- **Nouveau** : Ajout de l'API REST.
- **Refonte** : Interface GUI entièrement recodée pour plus de fluidité.

## v1.4.2
- **Correctif** : Problème de compatibilité avec Paper 1.20.4.
- **Amélioration** : Messages d'erreur plus clairs dans la console.

## v1.4.0
- **Nouveau** : Système d'[Événements de Marché](/fr/features/events).
- **Nouveau** : Support des BossBars pour les événements.

## v1.0.0
- Sortie initiale publique.
- Système de prix dynamique de base.
- Support Vault.
