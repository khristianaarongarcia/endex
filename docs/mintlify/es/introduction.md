## ¿Qué es The Endex?

**The Endex** convierte tu servidor de Minecraft en una economía de mercado donde los precios de los objetos cambian dinámicamente basándose en la oferta y demanda. Los jugadores compran, venden, guardan y comercian mientras los precios reaccionan en tiempo real.

## Características Principales

<CardGroup cols={2}>
  <Card title="Precios Dinámicos" icon="chart-line">
    Los precios fluctúan según la oferta/demanda y eventos del mercado.
  </Card>
  <Card title="Panel Web" icon="browser">
    Panel web integrado con gráficos en tiempo real e interfaz de trading.
  </Card>
  <Card title="Tenencias Virtuales" icon="warehouse">
    Guarda objetos sin ocupar espacio en el inventario.
  </Card>
  <Card title="Sistema de Inversiones" icon="piggy-bank">
    Bloquea objetos para ganar APR a lo largo del tiempo.
  </Card>
  <Card title="Eventos del Mercado" icon="calendar">
    Crea multiplicadores de precios temporales para eventos.
  </Card>
  <Card title="API Completa" icon="code">
    API REST y Java para extensiones de plugins.
  </Card>
</CardGroup>

## Inicio Rápido

<Steps>
  <Step title="Instalar Dependencias">
    Asegúrate de tener Vault + un plugin de economía (EssentialsX)
  </Step>
  <Step title="Descargar The Endex">
    Obtén el JAR desde SpigotMC o Modrinth
  </Step>
  <Step title="Reiniciar Servidor">
    Coloca el JAR en `/plugins` y reinicia
  </Step>
  <Step title="Configurar">
    Edita `config.yml` para tus necesidades
  </Step>
</Steps>

¿Listo para comenzar? Consulta la guía de [Instalación](/es/getting-started/installation).
