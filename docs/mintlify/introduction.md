**The Endex** is a dynamic, player-driven market plugin for Minecraft servers.

It provides:

- **Live item pricing** that moves based on supply and demand
- **Holdings** (virtual storage) so players can trade without inventory juggling
- **Delivery queue** so purchases never disappear when storage is full
- **Market events** for spikes, crashes, and server-wide moments
- **Web dashboard + REST API** for fast trading and integration

## Key highlights

| Feature | Description |
| --- | --- |
| Dynamic Pricing | Real supply/demand economics |
| World Storage Scanning | Prices can react to stored items (optional) |
| Virtual Holdings | Store purchased items with P/L tracking |
| Web Dashboard | Trade from your browser with live updates |
| Market GUI | In-game trading interface |
| Delivery System | Never lose overflow items |
| Addon Support | Extend functionality with addons |
| Investments | APR-based investment certificates |
