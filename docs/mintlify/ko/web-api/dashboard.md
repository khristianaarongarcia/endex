# 웹 대시보드

The Endex에는 플레이어가 게임 밖에서 경제를 추적할 수 있도록 아름답고 반응형인 대시보드를 제공하는 내장 웹 서버가 포함되어 있습니다.

## 기능

- **실시간 데이터**: 가격은 WebSocket을 통해 즉시 업데이트됩니다.
- **기록 차트**: 24시간, 7일 또는 30일 동안의 가격 추세를 시각화합니다.
- **최고 상승/하락**: 어떤 아이템이 가장 좋은/나쁜 성과를 내고 있는지 빠르게 확인합니다.
- **반응형**: 모바일 장치와 데스크톱에서 완벽하게 작동합니다.

## 구성

대시보드를 활성화하려면 `config.yml`을 편집하세요:

```yaml
web:
  enabled: true
  port: 8080 # 웹 서버 포트
  host: "0.0.0.0" # 모든 인터페이스에서 수신 대기
  public-url: "http://your-server.com:8080" # 링크용 URL
```

### 네트워킹

집에서 서버를 호스팅하는 경우 포트 `8080`(또는 선택한 포트)이 방화벽에서 열려 있고 포트 포워딩되어 있는지 확인하세요.

## 액세스

활성화하고 서버를 재시작한 후 브라우저에서 대시보드에 액세스하세요:

`http://<서버 IP>:8080`

## 사용자 정의

`plugins/TheEndex/web/` 폴더 내의 HTML/CSS/JS 파일을 편집하여 대시보드 모양을 사용자 정의할 수 있습니다.

- `index.html`: 메인 페이지.
- `css/style.css`: 스타일.
- `js/app.js`: 프론트엔드 로직.

*참고: 고급 사용자 정의를 위해서는 웹 개발에 대한 기본 지식이 권장됩니다.*
