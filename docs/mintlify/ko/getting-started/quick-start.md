# 빠른 시작

이 가이드는 The Endex를 사용하여 첫 번째 동적 경제를 설정하는 데 도움을 줍니다.

## 1. 시장에 아이템 추가

아이템에 동적 가격을 설정하려면 The Endex에 추가해야 합니다.

### 게임 내에서

추가하려는 아이템을 주 손에 들고 다음을 입력하세요:

```bash
/endex admin additem <시작 가격>
```

예를 들어, 기본 가격 100으로 다이아몬드를 추가하려면:

```bash
/endex admin additem 100
```

### 구성을 통해

`items.yml`에서 수동으로 아이템을 추가할 수도 있습니다:

```yaml
DIAMOND:
  base-price: 100.0
  min-price: 10.0
  max-price: 1000.0
  volatility: 0.5
```

## 2. 시장 표지판 만들기 (선택 사항)

표지판을 사용하여 가격을 표시하려면:

1. 표지판을 설치합니다.
2. 첫 번째 줄에 `[Endex]`라고 씁니다.
3. 두 번째 줄에 아이템 이름(예: `DIAMOND`)을 씁니다.

표지판은 현재 가격으로 자동 업데이트됩니다.

## 3. GUI 메뉴 열기

플레이어는 메인 메뉴에서 모든 시장 아이템과 추세를 확인할 수 있습니다:

```bash
/market
```

## 4. 거래 시뮬레이션

가격 변동을 테스트하기 위해 구매 및 판매를 시뮬레이션할 수 있습니다:

- **구매**하면 가격이 상승합니다 (수요 증가).
- **판매**하면 가격이 하락합니다 (공급 증가).

아이템을 몇 스택 구매해 보고 가격이 어떻게 변하는지 관찰해 보세요!
