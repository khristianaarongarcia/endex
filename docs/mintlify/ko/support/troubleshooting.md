# 문제 해결

The Endex에 문제가 발생하면 다음 진단 단계를 따르세요.

## 일반적인 문제

### 플러그인이 시작되지 않음 (/plugins에서 빨간색)
- **원인**: 종속성 누락.
- **해결책**: **Vault** 및 경제 플러그인(EssentialsX)이 설치되어 있는지 확인하세요. 시작 로그에서 "Dependency not found"를 확인하세요.

### 명령어 실행 시 "An internal error occurred..." 표시됨
- **원인**: 플러그인 버그 또는 구성 손상.
- **해결책**:
  1. 콘솔에서 "Stack Trace"(오류 텍스트 블록)를 확인하세요.
  2. 온라인 YAML 검사기로 YAML 파일(config, items)을 확인하세요. 단순한 들여쓰기 오류로 인해 모든 것이 깨질 수 있습니다.
  3. 오류가 지속되면 Discord 또는 GitHub에 보고하세요.

### 가격이 변하지 않음
- **원인**: 변동성이 너무 낮거나 거래가 없음.
- **해결책**:
  - `items.yml`의 `volatility`가 > 0인지 확인하세요.
  - 시장이 "폐장" 또는 "동결"되지 않았는지 확인하세요(해당 기능이 활성화된 경우).
  - 테스트 거래를 수행하여 무언가 움직이는지 확인하세요.

### 웹 대시보드에 "Disconnected" 표시됨
- **원인**: WebSocket 또는 포트 문제.
- **해결책**:
  - 포트가 열려 있는지 확인하세요(TCP).
  - 프록시(Nginx/Apache)를 사용하는 경우 WebSocket 지원이 구성되어 있는지 확인하세요.
  - `config.yml`에 구성된 URL을 확인하세요.

## 도움 요청

여기서 해결책을 찾을 수 없는 경우:

1. **로그**: 콘솔에서 관련 로그를 복사하세요.
2. **구성**: 구성 파일 사본을 준비하세요(비밀번호 제외).
3. **연락**: 지원 Discord 서버에 가입하거나 GitHub에서 이슈를 여세요.

우리가 도울 수 있도록 가능한 한 많은 세부 정보를 제공해 주세요!
