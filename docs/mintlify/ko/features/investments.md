# 투자

단순한 아이템 매매를 넘어, The Endex는 고급 서버를 위한 더 복잡한 투자 메커니즘을 제공합니다.

## 상장지수펀드 (ETF)

*곧 출시 예정 / 개발 중인 기능*

플레이어가 아이템 바구니(예: 철, 금, 다이아몬드, 석탄으로 구성된 "광업 펀드")에 투자할 수 있게 합니다. 펀드의 가치는 기초 아이템의 가중 평균을 따릅니다.

## 배당금

아이템을 (가상의) "주식"으로 설정하는 경우 정기적인 배당금을 설정할 수 있습니다.

```yaml
# items.yml 예시
SERVER_SHARE:
  base-price: 1000.0
  dividend:
    enabled: true
    amount: 10.0 # 주당 $10 지급
    interval: 86400 # 24시간마다
```

[포트폴리오](/ko/features/holdings)에 이 아이템을 소유한 플레이어는 각 간격마다 자동으로 돈을 받습니다.

## 공매도 (Short Selling)

*고급 기능*

플레이어가 가격 하락에 베팅할 수 있게 합니다.
1. 플레이어는 아이템을 "빌려서" 현재 가격(예: $100)에 팝니다.
2. 빚을 갚기 위해 나중에 다시 사야 합니다.
3. 가격이 $80로 떨어지면 다시 사서 차액($20 이익)을 챙깁니다.
4. 가격이 $120로 오르면 $20 손실을 봅니다.

이 기능은 경험이 부족한 플레이어에게 위험할 수 있으므로 명시적으로 활성화해야 합니다.
