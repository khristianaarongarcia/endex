# 시장 이벤트

시장 이벤트는 경제에 예측 불가능성과 흥미를 더합니다. 이는 갑작스러운 가격 급등(강세장)이나 폭락(약세장/붕괴)을 유발할 수 있습니다.

## 이벤트 유형

### 1. 시장 붕괴 (Market Crash)
모든 가격 또는 특정 그룹의 아이템 가격이 크게 하락합니다.
- **원인**: 공황, 과잉 생산 또는 무작위.
- **효과**: 가격이 X% 하락합니다.

### 2. 시장 호황 (Market Boom)
가격이 급격히 상승합니다.
- **원인**: 희소성, 높은 수요 또는 무작위.
- **효과**: 가격이 X% 상승합니다.

### 3. 특정 이벤트
단일 아이템 또는 카테고리에 영향을 미칩니다.
- *예: "새로운 금맥이 발견되었습니다!" -> 금 가격 하락.*
- *예: "밀 흉작입니다!" -> 빵 가격 상승.*

## 이벤트 구성

이벤트는 `events.yml`에서 구성됩니다.

```yaml
events:
  gold_rush:
    display-name: "골드 러시"
    message: "&6새로운 금맥이 발견되었습니다! 금 가격이 하락하고 있습니다!"
    items: [GOLD_INGOT, GOLD_BLOCK]
    price-multiplier: 0.5 # 가격 반토막
    duration: 3600 # 1시간 지속
    chance: 0.01 # 주기당 1% 확률
```

## 트리거

### 자동
플러그인은 주기적으로(구성 가능) 확률(`chance`)에 따라 이벤트를 트리거할지 확인합니다.

### 수동 (관리자)
관리자는 이벤트를 강제로 시작할 수 있습니다:

```bash
/endex admin event start <이벤트 이름>
```

## 알림

이벤트가 시작되면 공지할 수 있습니다:
- 채팅으로.
- 보스바(BossBar)를 통해.
- 화면 타이틀을 통해.

이는 플레이어에게 거래 기회가 있음을 경고합니다.
