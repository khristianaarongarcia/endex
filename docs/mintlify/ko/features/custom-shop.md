# 커스텀 상점

The Endex는 기존 상점을 완전히 대체하는 것이 아니라 가격 엔진으로 작동하도록 설계되었습니다.

## GUI 통합

The Endex는 가격과 추세를 보기 위한 자체 그래픽 사용자 인터페이스(GUI)를 제공하지만, 물리적 거래의 경우 다른 시스템과 통합하거나 명령어를 통한 직접 구매/판매를 허용하는 경우가 많습니다.

### 메인 메뉴

`/market` 명령어는 메인 메뉴를 열어 플레이어가 다음을 수행할 수 있게 합니다:

- 나열된 모든 아이템 확인.
- 현재 가격, 변동률(24시간), 추세(상승/하락) 확인.
- 포트폴리오(보유 자산) 액세스.

## 물리적 상점 만들기

The Endex 가격을 사용하는 물리적 상점을 만들 수 있습니다.

### 표지판 (Signs)

The Endex는 동적 거래 표지판을 지원합니다.

**생성 형식:**
1. `[Endex]`
2. `BUY` 또는 `SELL`
3. `<아이템 이름>`
4. `<수량>`

**예시:**
```text
[Endex]
BUY
DIAMOND
1
```

이렇게 하면 플레이어가 현재 시장 가격으로 다이아몬드 1개를 구매할 수 있는 표지판이 생성됩니다. 표지판에 표시된 가격은 자동으로 업데이트됩니다.

### Citizens 통합 (NPC)

**Citizens** 플러그인이 있는 경우 상인 특성을 만들 수 있습니다.

```bash
/npc create Trader --type villager
/trait endex-trader
```

*(참고: 이 기능은 The Endex 버전 및 설치된 애드온에 따라 다를 수 있습니다)*

## 빠른 구매/판매 명령어

플레이어는 GUI 없이 빠르게 거래하기 위해 명령어를 사용할 수도 있습니다:

- `/endex buy <아이템> <수량>`
- `/endex sell <아이템> <수량>`
- `/endex sellall`: 호환되는 인벤토리 내의 모든 것을 시장에 판매합니다.

## 메시지 사용자 정의

모든 거래 관련 메시지(성공, 자금 부족, 인벤토리 가득 참)는 `messages.yml`에서 구성할 수 있습니다.
