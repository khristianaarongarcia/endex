# 権限

これらの権限を使用して、The Endex の機能へのアクセスを管理します。

## ユーザー権限

これらの権限は、デフォルトでプレイヤーに推奨されます。

| 権限 | 説明 | 推奨 |
|------|------|------|
| `endex.use` | プラグインの使用とGUIを開くことを許可します。 | **はい** |
| `endex.help` | ヘルプの表示を許可します。 | **はい** |
| `endex.price` | 価格の確認を許可します。 | **はい** |
| `endex.buy` | アイテムの購入を許可します。 | **はい** |
| `endex.sell` | アイテムの売却を許可します。 | **はい** |
| `endex.sellall` | `/endex sellall` の使用を許可します。 | **はい** |
| `endex.holdings` | ポートフォリオへのアクセスを許可します。 | **はい** |
| `endex.invest` | 投資（保有資産の購入）を許可します。 | **はい** |
| `endex.divest` | 投資の引き上げ（保有資産の売却）を許可します。 | **はい** |
| `endex.history` | 価格履歴の表示を許可します。 | **はい** |

## 管理者権限

⚠️ **警告**: これらの権限は信頼できる管理者にのみ付与してください。

| 権限 | 説明 |
|------|------|
| `endex.admin.*` | すべての管理者権限を付与します。 |
| `endex.admin.reload` | 設定のリロードを許可します。 |
| `endex.admin.additem` | 市場へのアイテムの追加を許可します。 |
| `endex.admin.removeitem` | アイテムの削除を許可します。 |
| `endex.admin.setprice` | 価格の操作を許可します。 |
| `endex.admin.event` | イベントの開始/停止を許可します。 |
| `endex.admin.debug` | デバッグ情報の表示を許可します。 |
| `endex.admin.update` | 更新通知を受け取ります。 |

## 特別な権限

| 権限 | 説明 |
|------|------|
| `endex.bypass.tax` | プレイヤーは保有資産に対する税金を支払いません。 |
| `endex.bypass.limit` | プレイヤーには保有資産の制限がありません。 |
