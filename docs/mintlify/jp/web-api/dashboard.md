# Webダッシュボード

The Endex には、プレイヤーがゲーム外で経済を追跡できるように、美しくレスポンシブなダッシュボードを提供する組み込みWebサーバーが含まれています。

## 機能

- **リアルタイムデータ**: 価格はWebSocket経由で即座に更新されます。
- **履歴チャート**: 24時間、7日間、または30日間の価格トレンドを視覚化します。
- **トップ上昇/下落**: どのアイテムが最も良い/悪いパフォーマンスを示しているかを素早く確認できます。
- **レスポンシブ**: モバイルデバイスとデスクトップで完璧に動作します。

## 設定

ダッシュボードを有効にするには、`config.yml` を編集します：

```yaml
web:
  enabled: true
  port: 8080 # Webサーバーのポート
  host: "0.0.0.0" # すべてのインターフェースでリッスン
  public-url: "http://your-server.com:8080" # リンク用URL
```

### ネットワーク

自宅でサーバーをホストしている場合は、ポート `8080`（または選択したポート）がファイアウォールで開放され、転送（ポートフォワーディング）されていることを確認してください。

## アクセス

有効化してサーバーを再起動したら、ブラウザからダッシュボードにアクセスします：

`http://<サーバーのIP>:8080`

## カスタマイズ

`plugins/TheEndex/web/` フォルダ内のHTML/CSS/JSファイルを編集することで、ダッシュボードの外観をカスタマイズできます。

- `index.html`: メインページ。
- `css/style.css`: スタイル。
- `js/app.js`: フロントエンドロジック。

*注: 高度なカスタマイズには、Web開発の基本的な知識が推奨されます。*
