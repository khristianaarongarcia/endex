# The Endex

**The Endex** は、リアルな株式市場のメカニズム、価格変動、投資システムをゲーム内経済にもたらす、Minecraftサーバー向けの高度な動的経済プラグインです。

## なぜ The Endex を選ぶのか？

- **リアリズム**: 実際の市場経済をシミュレートします。
- **エンゲージメント**: プレイヤーが市場の動向を注視し続けるようにします。
- **カスタマイズ**: サーバーのニーズに合わせて高度に設定可能です。
- **パフォーマンス**: ラグなしで大規模な経済システムを処理できるように最適化されています。

## 言語を変更する

このドキュメントは **10言語** で利用可能です。言語を切り替えるには：

1. ドキュメントのヘッダー/ナビゲーションで **言語セレクター** を探します
2. クリックして利用可能な言語を表示します
3. お好みの言語を選択します

| 言語 | コード |
| --- | --- |
| 🇺🇸 英語 | `en` |
| 🇨🇳 中国語（簡体字） | `cn` |
| 🇪🇸 スペイン語 | `es` |
| 🇫🇷 フランス語 | `fr` |
| 🇩🇪 ドイツ語 | `de` |
| 🇯🇵 日本語 | `jp` |
| 🇰🇷 韓国語 | `ko` |
| 🇧🇷 ポルトガル語（ブラジル） | `pt-BR` |
| 🇷🇺 ロシア語 | `ru` |
| 🇵🇱 ポーランド語 | `pl` |

<Tip>
ゲーム内プラグインも複数の言語をサポートしています！`config.yml` の `language: jp` でお好みの言語を設定してください（上の表の言語コードを使用）。
</Tip>
