# トラブルシューティング

The Endex で問題が発生した場合は、以下の診断手順に従ってください。

## 一般的な問題

### プラグインが起動しない (/plugins で赤色)
- **原因**: 依存関係の欠落。
- **解決策**: **Vault** と経済プラグイン（EssentialsX）がインストールされていることを確認してください。起動ログで "Dependency not found" を確認してください。

### コマンド実行時に "An internal error occurred..." が表示される
- **原因**: プラグインのバグまたは設定の破損。
- **解決策**:
  1. コンソールで "Stack Trace"（エラーテキストブロック）を確認してください。
  2. オンラインYAMLバリデーターでYAMLファイル（config, items）を確認してください。単純なインデントエラーですべてが壊れる可能性があります。
  3. エラーが続く場合は、DiscordまたはGitHubで報告してください。

### 価格が変わらない
- **原因**: ボラティリティが低すぎるか、取引がない。
- **解決策**:
  - `items.yml` の `volatility` が > 0 であることを確認してください。
  - 市場が「クローズ」または「凍結」されていないことを確認してください（その機能がアクティブな場合）。
  - テスト取引を行って、何かが動くかどうかを確認してください。

### Webダッシュボードに "Disconnected" と表示される
- **原因**: WebSocketまたはポートの問題。
- **解決策**:
  - ポートが開放されていることを確認してください（TCP）。
  - プロキシ（Nginx/Apache）を使用している場合は、WebSocketサポートが設定されていることを確認してください。
  - `config.yml` で設定されたURLを確認してください。

## 助けを求める

ここで解決策が見つからない場合：

1. **ログ**: コンソールから関連するログをコピーしてください。
2. **設定**: 設定ファイルのコピーを準備してください（パスワードを除く）。
3. **連絡**: サポートDiscordサーバーに参加するか、GitHubでIssueを開いてください。

私たちが助けられるように、できるだけ多くの詳細を提供してください！
