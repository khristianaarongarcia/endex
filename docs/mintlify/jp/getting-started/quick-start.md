# クイックスタート

このガイドは、The Endex を使用して最初の動的経済を設定するのに役立ちます。

## 1. 市場へのアイテムの追加

アイテムに動的な価格を設定するには、The Endex に追加する必要があります。

### ゲーム内

追加したいアイテムをメインハンドに持ち、次のように入力します：

```bash
/endex admin additem <開始価格>
```

例えば、基本価格100でダイヤモンドを追加するには：

```bash
/endex admin additem 100
```

### 設定経由

`items.yml` で手動でアイテムを追加することもできます：

```yaml
DIAMOND:
  base-price: 100.0
  min-price: 10.0
  max-price: 1000.0
  volatility: 0.5
```

## 2. 市場看板の作成（オプション）

看板を使用して価格を表示したい場合：

1. 看板を設置します。
2. 1行目に `[Endex]` と書きます。
3. 2行目にアイテム名（例：`DIAMOND`）を書きます。

看板は現在の価格で自動的に更新されます。

## 3. GUIメニューを開く

プレイヤーはメインメニューからすべての市場アイテムとそのトレンドを確認できます：

```bash
/market
```

## 4. 取引のシミュレーション

価格変動をテストするために、購入と売却をシミュレートできます：

- **購入** すると価格が上がります（需要増）。
- **売却** すると価格が下がります（供給増）。

アイテムを数スタック購入してみて、価格がどのように変化するか観察してください！
