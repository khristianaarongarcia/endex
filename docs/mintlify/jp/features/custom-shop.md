# カスタムショップ

The Endex は、既存のショップを完全に置き換えるのではなく、価格エンジンとして機能するように設計されています。

## GUI統合

The Endex は価格とトレンドを表示するための独自のグラフィカルユーザーインターフェース（GUI）を提供しますが、物理的な取引については、他のシステムと統合したり、コマンドによる直接購入/売却を許可したりすることがよくあります。

### メインメニュー

`/market` コマンドはメインメニューを開き、プレイヤーは以下を行うことができます：

- リストされているすべてのアイテムを確認する。
- 現在の価格、変化率（24時間）、トレンド（上昇/下降）を確認する。
- ポートフォリオ（保有資産）にアクセスする。

## 物理ショップの作成

The Endex の価格を使用する物理ショップを作成できます。

### 看板（Signs）

The Endex は動的取引看板をサポートしています。

**作成フォーマット:**
1. `[Endex]`
2. `BUY` または `SELL`
3. `<アイテム名>`
4. `<数量>`

**例:**
```text
[Endex]
BUY
DIAMOND
1
```

これにより、プレイヤーが現在の市場価格でダイヤモンドを1個購入できる看板が作成されます。看板に表示される価格は自動的に更新されます。

### Citizens統合（NPC）

**Citizens** プラグインをお持ちの場合、トレーダー特性を作成できます。

```bash
/npc create Trader --type villager
/trait endex-trader
```

*(注: この機能は The Endex のバージョンとインストールされているアドオンに依存します)*

## クイック購入/売却コマンド

プレイヤーはGUIなしで素早く取引するためにコマンドを使用することもできます：

- `/endex buy <アイテム> <数量>`
- `/endex sell <アイテム> <数量>`
- `/endex sellall`: 互換性のあるインベントリ内のすべてを市場に売却します。

## メッセージのカスタマイズ

すべての取引関連メッセージ（成功、資金不足、インベントリ満杯）は `messages.yml` で設定可能です。
