# 投資

アイテムの単純な売買を超えて、The Endex は上級サーバー向けのより複雑な投資メカニズムを提供します。

## 投資信託 (ETF)

*近日公開 / 開発中の機能*

プレイヤーがアイテムのバスケット（例：鉄、金、ダイヤモンド、石炭で構成される「鉱業ファンド」）に投資できるようにします。ファンドの価値は、基礎となるアイテムの加重平均に従います。

## 配当

アイテムを（仮想的な）「株式」として設定する場合、定期的な配当を設定できます。

```yaml
# items.yml の例
SERVER_SHARE:
  base-price: 1000.0
  dividend:
    enabled: true
    amount: 10.0 # 1株あたり$10支払う
    interval: 86400 # 24時間ごと
```

[ポートフォリオ](/jp/features/holdings)にこのアイテムを所有しているプレイヤーは、各間隔で自動的にお金を受け取ります。

## 空売り (Short Selling)

*高度な機能*

プレイヤーが価格の下落に賭けることを可能にします。
1. プレイヤーはアイテムを「借りて」、現在の価格（例：$100）で売ります。
2. 借金を返すために、後で買い戻す必要があります。
3. 価格が$80に下がった場合、買い戻して差額（$20の利益）を保持します。
4. 価格が$120に上がった場合、$20の損失になります。

この機能は、経験の浅いプレイヤーにとってリスクがあるため、明示的に有効にする必要があります。
