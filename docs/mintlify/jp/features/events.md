# 市場イベント

市場イベントは経済に予測不可能性と興奮をもたらします。これらは突然の価格高騰（強気市場）や暴落（弱気市場/クラッシュ）を引き起こす可能性があります。

## イベントの種類

### 1. 市場暴落 (Market Crash)
すべての価格、または特定のグループのアイテムの価格が大幅に下落します。
- **原因**: パニック、過剰生産、またはランダム。
- **効果**: 価格がX%下落します。

### 2. 市場好況 (Market Boom)
価格が急速に上昇します。
- **原因**: 希少性、高い需要、またはランダム。
- **効果**: 価格がX%上昇します。

### 3. 特定のイベント
単一のアイテムまたはカテゴリに影響します。
- *例: 「新しい金脈が発見されました！」 -> 金の価格が下落。*
- *例: 「小麦が不作です！」 -> パンの価格が上昇。*

## イベントの設定

イベントは `events.yml` で設定されます。

```yaml
events:
  gold_rush:
    display-name: "ゴールドラッシュ"
    message: "&6新しい金脈が発見されました！金の価格が下落しています！"
    items: [GOLD_INGOT, GOLD_BLOCK]
    price-multiplier: 0.5 # 価格が半減
    duration: 3600 # 1時間継続
    chance: 0.01 # サイクルごとに1%の確率
```

## トリガー

### 自動
プラグインは定期的に（設定可能）、確率（`chance`）に基づいてイベントをトリガーするかどうかを確認します。

### 手動 (管理者)
管理者はイベントを強制的に開始できます：

```bash
/endex admin event start <イベント名>
```

## 通知

イベントが始まると、アナウンスを行うことができます：
- チャットで。
- BossBar経由で。
- 画面上のタイトル経由で。

これはプレイヤーに取引の機会があることを警告します。
