# The Endex

**The Endex** é um plugin de economia dinâmica avançada para servidores Minecraft que introduz mecânicas realistas de mercado de ações, flutuações de preços e sistemas de investimento na economia do seu jogo.

## Por que escolher o The Endex?

- **Realismo**: Simula uma economia de mercado real.
- **Engajamento**: Mantém os jogadores observando as tendências do mercado.
- **Personalização**: Altamente configurável para atender às necessidades do seu servidor.
- **Desempenho**: Otimizado para lidar com grandes economias sem lag.
