# Кастомный магазин

The Endex разработан как движок ценообразования, а не обязательно как полная замена вашего существующего магазина.

## Интеграция GUI

Хотя The Endex предоставляет собственный графический интерфейс (GUI) для просмотра цен и трендов, вы можете захотеть интегрировать его с другими системами для физических транзакций или разрешить прямую покупку/продажу через команды.

### Главное меню

Команда `/market` открывает главное меню, где игроки могут:

- Видеть все перечисленные предметы.
- Видеть текущую цену, процентное изменение (24ч) и тренд (рост/падение).
- Получить доступ к своему портфелю (holdings).

## Создание физических магазинов

Вы можете создавать физические магазины, использующие цены The Endex.

### Таблички (Signs)

The Endex поддерживает динамические торговые таблички.

**Формат создания:**
1. `[Endex]`
2. `BUY` или `SELL`
3. `<Название предмета>`
4. `<Количество>`

**Пример:**
```text
[Endex]
BUY
DIAMOND
1
```

Это создаст табличку, где игроки могут купить 1 Алмаз по текущей рыночной цене. Цена на табличке будет обновляться автоматически.

### Интеграция с Citizens (NPC)

Если у вас есть плагин **Citizens**, вы можете создавать торговцев.

```bash
/npc create Trader --type villager
/trait endex-trader
```

*(Примечание: Эта функция может зависеть от версии The Endex и установленных аддонов)*

## Команды быстрой покупки/продажи

Игроки также могут использовать команды для быстрой торговли без GUI:

- `/endex buy <предмет> <количество>`
- `/endex sell <предмет> <количество>`
- `/endex sellall`: Продает все в инвентаре, что совместимо с рынком.

## Настройка сообщений

Все сообщения, связанные с транзакциями (успех, недостаточно средств, инвентарь полон), можно настроить в `messages.yml`.
