# Система портфеля (Holdings)

Система **Holdings** (Портфель) позволяет игрокам инвестировать в предметы без необходимости физически хранить их в инвентаре. Это похоже на покупку акций или фьючерсных контрактов в реальном мире.

## Как это работает

Вместо покупки физического "Алмаза", игрок покупает "Долю в Алмазе".

1. Игрок платит текущую рыночную цену.
2. Доля сохраняется в его виртуальном портфеле.
3. Игрок не получает физический предмет.
4. Позже игрок может продать долю по новой рыночной цене.

### Преимущества

- **Нет управления инвентарем**: Игроки могут торговать тысячами единиц без сундуков.
- **Чистая спекуляция**: Отлично подходит для игроков, которые хотят быть трейдерами, не собирая ресурсы.
- **Защита**: Активы в портфеле нельзя украсть, сжечь или потерять при смерти.

## Команды портфеля

- `/holdings`: Открывает GUI портфеля для просмотра инвестиций.
- `/endex invest <предмет> <количество>`: Покупает активы (виртуально).
- `/endex divest <предмет> <количество>`: Продает активы (виртуально).

## Конфигурация

Вы можете включить или отключить эту систему в `config.yml`:

```yaml
holdings:
  enabled: true
  max-holdings-per-item: 10000 # Лимит на игрока
  tax-rate: 0.05 # 5% налог на прибыль с портфеля
```

## Налог на прирост капитала

Чтобы предотвратить слишком быструю инфляцию экономики из-за спекуляций, вы можете ввести налог (`tax-rate`) на продажу активов. Этот налог вычитается из прибыли или общей суммы при продаже.
