# Инвестиции

Помимо простой покупки и продажи предметов, The Endex предлагает более сложные инвестиционные механики для продвинутых серверов.

## ETF (Биржевые фонды)

*Скоро / Функция в разработке*

Позволяет игрокам инвестировать в корзину предметов (например, "Шахтерский фонд", содержащий Железо, Золото, Алмаз, Уголь). Стоимость фонда следует за средневзвешенной стоимостью базовых предметов.

## Дивиденды

Если вы настроите предмет действовать как "Акция" (виртуальная), вы можете настроить регулярные выплаты дивидендов.

```yaml
# Пример в items.yml
SERVER_SHARE:
  base-price: 1000.0
  dividend:
    enabled: true
    amount: 10.0 # Платит $10 за акцию
    interval: 86400 # Каждые 24 часа
```

Игроки, владеющие этим предметом в своем [Портфеле](/ru/features/holdings), будут автоматически получать деньги каждый интервал.

## Короткая продажа (Short Selling)

*Продвинутая функция*

Позволяет игрокам делать ставки на падение цены.
1. Игрок "занимает" предмет и продает его по текущей цене (например, $100).
2. Он должен выкупить его позже, чтобы вернуть долг.
3. Если цена упадет до $80, он выкупает его и оставляет разницу ($20 прибыли).
4. Если цена вырастет до $120, он теряет $20.

Это должно быть включено явно, так как это может быть рискованно для неопытных игроков.
