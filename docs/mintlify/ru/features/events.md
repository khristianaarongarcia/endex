# Рыночные события

Рыночные события добавляют непредсказуемость и азарт в экономику. Они могут вызвать внезапные скачки цен (Бычий рынок) или падения (Медвежий рынок/Крах).

## Типы событий

### 1. Крах рынка (Market Crash)
Значительное падение цен на все предметы или определенную группу.
- **Причина**: Паника, перепроизводство или случайность.
- **Эффект**: Цены падают на X%.

### 2. Бум рынка (Market Boom)
Быстрый рост цен.
- **Причина**: Дефицит, высокий спрос или случайность.
- **Эффект**: Цены растут на X%.

### 3. Специфические события
Влияют на один предмет или категорию.
- *Пример: "Найдена новая золотая жила!" -> Цена Золота падает.*
- *Пример: "Чума на пшеничных полях!" -> Цена Хлеба растет.*

## Настройка событий

События настраиваются в `events.yml`.

```yaml
events:
  gold_rush:
    display-name: "Золотая лихорадка"
    message: "&6Найдена новая золотая жила! Цена на золото падает!"
    items: [GOLD_INGOT, GOLD_BLOCK]
    price-multiplier: 0.5 # Цена уменьшается вдвое
    duration: 3600 # Длится 1 час
    chance: 0.01 # 1% шанс за цикл
```

## Триггеры

### Автоматический
Плагин периодически проверяет (настраивается), нужно ли запускать событие на основе `chance`.

### Ручной (Админ)
Администраторы могут принудительно запустить событие:

```bash
/endex admin event start <название_события>
```

## Уведомления

Когда событие начинается, вы можете объявить об этом:
- В чате.
- Через BossBar.
- Через Титул на экране.

Это предупреждает игроков о торговых возможностях.
