# REST API

The Endex предоставляет RESTful API, позволяющий внешним разработчикам взаимодействовать с рыночными данными. Это полезно для создания сайтов статистики, ботов Discord или мобильных приложений.

## Эндпоинт (Базовый URL)

`http://<IP Сервера>:<Порт>/api/v1`

## Эндпоинты

### Получить все предметы

`GET /items`

Возвращает список всех отслеживаемых предметов.

**Ответ:**
```json
[
  {
    "id": "DIAMOND",
    "name": "Алмаз",
    "price": 105.50,
    "basePrice": 100.0,
    "trend": "UP",
    "change": 5.5
  },
  {
    "id": "GOLD_INGOT",
    "name": "Золотой слиток",
    "price": 48.20,
    "basePrice": 50.0,
    "trend": "DOWN",
    "change": -3.6
  }
]
```

### Получить конкретный предмет

`GET /items/{id}`

**Пример:** `GET /items/DIAMOND`

**Ответ:**
```json
{
  "id": "DIAMOND",
  "name": "Алмаз",
  "price": 105.50,
  "history": [
    {"time": 1620000000, "price": 100.0},
    {"time": 1620003600, "price": 102.0},
    {"time": 1620007200, "price": 105.5}
  ]
}
```

### Получить статус рынка

`GET /status`

Возвращает глобальный статус рынка (активные события и т.д.).

**Ответ:**
```json
{
  "status": "OPEN",
  "activeEvents": [],
  "lastUpdate": 1620007200
}
```

## Аутентификация

В настоящее время API является публичным и доступным только для чтения по умолчанию. Будущие версии будут включать аутентификацию по API-ключу для операций записи (POST/PUT).

## Ограничение скорости (Rate Limiting)

Жесткого ограничения скорости нет, но, пожалуйста, используйте здравый смысл и кэшируйте результаты, если делаете много запросов.
