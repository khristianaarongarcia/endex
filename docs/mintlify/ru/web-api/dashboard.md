# Веб-панель

The Endex включает встроенный веб-сервер, который предоставляет красивую и адаптивную панель для игроков, чтобы отслеживать экономику вне игры.

## Возможности

- **Данные в реальном времени**: Цены обновляются мгновенно через WebSockets.
- **Исторические графики**: Визуализируйте тренды цен за последние 24ч, 7д или 30д.
- **Лидеры роста/падения**: Быстро смотрите, какие предметы показывают лучшие/худшие результаты.
- **Адаптивность**: Отлично работает на мобильных устройствах и десктопах.

## Конфигурация

Чтобы включить панель, отредактируйте ваш `config.yml`:

```yaml
web:
  enabled: true
  port: 8080 # Порт веб-сервера
  host: "0.0.0.0" # Слушать на всех интерфейсах
  public-url: "http://your-server.com:8080" # URL для ссылок
```

### Сеть

Если вы хостите дома, убедитесь, что порт `8080` (или выбранный вами) открыт в вашем брандмауэре и проброшен (port forwarded).

## Доступ

После включения и перезапуска сервера, получите доступ к панели в браузере:

`http://<IP Сервера>:8080`

## Кастомизация

Вы можете настроить внешний вид панели, редактируя HTML/CSS/JS файлы внутри папки `plugins/TheEndex/web/`.

- `index.html`: Главная страница.
- `css/style.css`: Стили.
- `js/app.js`: Логика фронтенда.

*Примечание: Для продвинутой кастомизации рекомендуются базовые знания веб-разработки.*
