# Система аддонов

The Endex поддерживает модульную систему аддонов, позволяющую добавлять специфические функции без перегрузки основного плагина.

## Официальные аддоны

### 1. CryptoAddon
Добавляет виртуальные криптовалюты (Bitcoin, Ethereum и т.д.) на рынок.
- **Особенности**: Цены на основе реального API (CoinGecko), отдельные крипто-кошельки.
- **Установка**: Поместите `Endex-Crypto.jar` в `plugins/TheEndex/addons/`.

### 2. DiscordLink
Подключает ваш рынок к вашему серверу Discord.
- **Особенности**: Слэш-команды в Discord (`/price`), оповещения о крахе в каналах, синхронизация ролей на основе богатства.

## Создание аддона

Разработчики могут создавать свои собственные аддоны, расширяя класс `EndexAddon`.

### Базовая структура

```java
public class MyCustomAddon extends EndexAddon {
    
    @Override
    public void onEnable() {
        getLogger().info("Мой аддон активирован!");
    }
    
    @Override
    public void onDisable() {
        // Очистка
    }
}
```

### Загрузка

Аддоны должны быть помещены в папку `plugins/TheEndex/addons/`. Они загружаются автоматически при запуске основного плагина.
