# Быстрый старт

Это руководство поможет вам настроить вашу первую динамическую экономику с использованием The Endex.

## 1. Добавление предметов на рынок

Чтобы предмет имел динамическую цену, он должен быть добавлен в The Endex.

### В игре

Держите предмет, который хотите добавить, в основной руке и введите:

```bash
/endex admin additem <начальная_цена>
```

Например, чтобы добавить Алмаз с базовой ценой 100:

```bash
/endex admin additem 100
```

### Через конфигурацию

Вы также можете добавить предметы вручную в `items.yml`:

```yaml
DIAMOND:
  base-price: 100.0
  min-price: 10.0
  max-price: 1000.0
  volatility: 0.5
```

## 2. Создание рыночных табличек (Опционально)

Если вы хотите использовать таблички для отображения цен:

1. Поставьте табличку.
2. Напишите `[Endex]` на первой строке.
3. Напишите название предмета (например, `DIAMOND`) на второй строке.

Табличка будет автоматически обновляться с текущей ценой.

## 3. Открытие меню GUI

Игроки могут видеть все предметы рынка и их тренды в главном меню:

```bash
/market
```

## 4. Симуляция торговли

Чтобы протестировать колебания цен, вы можете симулировать покупки и продажи:

- **Покупка** увеличит цену (рост спроса).
- **Продажа** уменьшит цену (рост предложения).

Попробуйте купить несколько стаков предмета и посмотрите, как изменится цена!
