# Руководство по установке

Следуйте этим шагам, чтобы установить The Endex на ваш сервер Minecraft.

## Предварительные требования

Перед началом убедитесь, что у вас есть:

- Сервер Minecraft под управлением **Spigot**, **Paper** или **Purpur** (версии 1.16 - 1.21+).
- Установленная **Java 17** или выше.
- Плагин экономики, совместимый с **Vault** (например, EssentialsX).
- Установленный **Vault**.
- **PlaceholderAPI** (необязательно, но рекомендуется для плейсхолдеров).

## Шаги установки

<Steps>
  <Step title="Скачать плагин">
    Скачайте последнюю версию `TheEndex.jar` с [SpigotMC](https://www.spigotmc.org/) или [Modrinth](https://modrinth.com/).
  </Step>
  <Step title="Установить файл">
    Поместите скачанный `.jar` файл в папку `plugins` вашего сервера.
  </Step>
  <Step title="Перезапустить сервер">
    Перезапустите сервер, чтобы загрузить плагин. Избегайте использования `/reload`.
  </Step>
  <Step title="Проверить установку">
    Проверьте консоль на наличие сообщения о запуске The Endex или введите `/plugins` в игре, чтобы увидеть The Endex зеленым цветом.
  </Step>
</Steps>

## Начальная конфигурация

После первого запуска в директории `plugins` будет создана папка `TheEndex`. Она содержит:

- `config.yml`: Основной файл конфигурации.
- `messages.yml`: Файл перевода игровых сообщений.
- `items.yml`: Конфигурация предметов и базовых цен.

Обратитесь к разделу [Конфигурация](/ru/reference/configuration) для получения подробной информации о настройке вашей установки.
