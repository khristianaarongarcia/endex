# Список изменений

## v1.5.7-dec1022 (Текущая версия)
- **Новое**: Многоязычная поддержка для веб-панели.
- **Новое**: Система автоматического перевода для файлов конфигурации.
- **Улучшение**: Оптимизация производительности базы данных.
- **Исправление**: Ошибка, при которой цены не сохранялись при принудительной остановке.

- **Исправление**: Проблемы совместимости с Paper 1.20.4.
