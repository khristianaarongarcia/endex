# Права доступа

Используйте эти права для управления доступом к функциям The Endex.

## Права пользователя

Рекомендуются для игроков по умолчанию.

| Право | Описание | Рекомендуется |
|-------|----------|---------------|
| `endex.use` | Позволяет использовать плагин и открывать GUI. | **Да** |
| `endex.help` | Позволяет видеть справку. | **Да** |
| `endex.price` | Позволяет проверять цены. | **Да** |
| `endex.buy` | Позволяет покупать предметы. | **Да** |
| `endex.sell` | Позволяет продавать предметы. | **Да** |
| `endex.sellall` | Позволяет использовать `/endex sellall`. | **Да** |
| `endex.holdings` | Позволяет доступ к портфелю. | **Да** |
| `endex.invest` | Позволяет инвестировать (покупать активы). | **Да** |
| `endex.divest` | Позволяет выводить инвестиции (продавать активы). | **Да** |
| `endex.history` | Позволяет видеть историю цен. | **Да** |

## Права администратора

⚠️ **Предупреждение**: Выдавайте эти права только доверенным администраторам.

| Право | Описание |
|-------|----------|
| `endex.admin.*` | Предоставляет все права админа. |
| `endex.admin.reload` | Позволяет перезагружать конфигурацию. |
| `endex.admin.additem` | Позволяет добавлять предметы на рынок. |
| `endex.admin.removeitem` | Позволяет удалять предметы. |
| `endex.admin.setprice` | Позволяет манипулировать ценами. |
| `endex.admin.event` | Позволяет запускать/останавливать события. |
| `endex.admin.debug` | Позволяет видеть отладочную информацию. |
| `endex.admin.update` | Получает уведомления об обновлениях. |

## Специальные права

| Право | Описание |
|-------|----------|
| `endex.bypass.tax` | Игрок не платит налоги на портфель. |
| `endex.bypass.limit` | У игрока нет лимита на портфель. |
