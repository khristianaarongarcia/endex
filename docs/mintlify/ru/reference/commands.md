# Команды

Здесь представлен полный список команд, доступных в The Endex.

## Команды игрока

Эти команды обычно доступны всем игрокам (в зависимости от прав).

| Команда | Алиас | Описание | Право |
|---------|-------|----------|-------|
| `/endex` | `/market` | Открывает главное меню рынка. | `endex.use` |
| `/endex help` | | Показывает справку. | `endex.help` |
| `/endex price <предмет>` | | Показывает текущую цену предмета. | `endex.price` |
| `/endex buy <предмет> <кол-во>` | | Покупает предметы по рыночной цене. | `endex.buy` |
| `/endex sell <предмет> <кол-во>` | | Продает предметы по рыночной цене. | `endex.sell` |
| `/endex sellall` | | Продает все совместимое в инвентаре. | `endex.sellall` |
| `/holdings` | `/portfolio` | Открывает меню портфеля. | `endex.holdings` |
| `/endex invest <предмет> <кол-во>` | | Покупает активы (виртуально). | `endex.invest` |
| `/endex divest <предмет> <кол-во>` | | Продает активы (виртуально). | `endex.divest` |

## Команды администратора

Эти команды ограничены администраторами и операторами.

| Команда | Описание | Право |
|---------|----------|-------|
| `/endex admin reload` | Перезагружает конфигурацию. | `endex.admin.reload` |
| `/endex admin additem <цена>` | Добавляет предмет в руке на рынок. | `endex.admin.additem` |
| `/endex admin removeitem <предмет>` | Удаляет предмет с рынка. | `endex.admin.removeitem` |
| `/endex admin setprice <предмет> <цена>` | Принудительно устанавливает цену предмета. | `endex.admin.setprice` |
| `/endex admin setvolatility <предмет> <знач>` | Изменяет волатильность предмета. | `endex.admin.setvolatility` |
| `/endex admin event start <событие>` | Принудительно запускает событие. | `endex.admin.event` |
| `/endex admin event stop` | Останавливает любое текущее событие. | `endex.admin.event` |
| `/endex admin debug` | Включает режим отладки в консоли. | `endex.admin.debug` |

## Аргументы

- `<предмет>`: Внутреннее имя предмета (например, `DIAMOND`, `IRON_INGOT`). Должно совпадать с `items.yml`.
- `<кол-во>`: Количество (целое число).
- `<цена>`: Цена (десятичное число).
