**The Endex** to dynamiczny, napędzany przez graczy plugin rynkowy dla serwerów Minecraft.

Oferuje:

- **Ceny przedmiotów na żywo**, które zmieniają się w zależności od podaży i popytu
- **Zasoby** (wirtualny magazyn), dzięki czemu gracze mogą handlować bez żonglowania ekwipunkiem
- **Kolejkę dostaw**, aby zakupy nigdy nie znikały, gdy magazyn jest pełny
- **Wydarzenia rynkowe** dla skoków cen, krachów i momentów obejmujących cały serwer
- **Dashboard Web + REST API** dla szybkiego handlu i integracji
- **Wsparcie PlaceholderAPI** dla scoreboardów, hologramów i list tab

## Kluczowe cechy

| Funkcja | Opis |
| --- | --- |
| Dynamiczne Ceny | Prawdziwa ekonomia podaży/popytu |
| Skanowanie Magazynów Świata | Ceny mogą reagować na przechowywane przedmioty (opcjonalne) |
| Wirtualne Zasoby | Przechowuj zakupione przedmioty ze śledzeniem zysków/strat |
| Dashboard Web | Handluj z przeglądarki z aktualizacjami na żywo |
| GUI Rynku | Interfejs handlu w grze |
| System Dostaw | Nigdy nie trać nadmiarowych przedmiotów |
| PlaceholderAPI | 30+ placeholderów dla zewnętrznych pluginów |
| Wsparcie Addonów | Rozszerz funkcjonalność za pomocą addonów |
| Inwestycje | Certyfikaty inwestycyjne oparte na APR |

## Co nowego w 1.5.7-dec1022

- **Tłumaczenie Dashboardu Web** — Integracja Google Translate dla 26+ języków
- **Poprawka GUI** — Naprawiono ArrayIndexOutOfBoundsException w panelu szczegółów rynku
- **Wskaźniki Wydajności** — Opcje konfiguracji pokazują teraz tagi `[PERF: LOW/MEDIUM/HIGH]`
- **Przetłumaczone Konfiguracje** — Wstępnie przetłumaczone pliki konfiguracyjne w 9 językach
- **Sprzedaż z Zasobów** — Sprzedawaj przedmioty bezpośrednio z wirtualnych zasobów (1.5.7)
- **Edytor Układu w Grze** — Wizualny edytor GUI dla własnych układów sklepów (1.5.6)
- **Wsparcie Arclight** — Pełna kompatybilność z serwerami hybrydowymi (1.5.7)
